Consolidate Explained

Consolidate is a Python & TypeScript based Web Application that fetches data I care about on a daily basis from Twitch, Youtube and the OldSchool Runescape Website.

Twitch Information: My followed channels that are currently live with their stream image, name, title, current viewer count and link to their livestream. This information will utilize a custom built sorting python library to sort based on view count or streamer username

Youtube Information: Videos posted by users I subscribe to with the video title, video views, video thumbnail, channel name and link to the video. I'd like to add functionality to group videos based on channel then sort those results by date posted. The possibilities for this integration are endless. Essentially, the goal is to create my own custom subscription box. For fun!

OldSchool Runescape Information: Playing around with Python webscraping this will gather and display the 5 most recently posted news articles taken from the oldschool.runescape.com website. This was an excuse to learn webscraping and fill out the page a bit more. I'm contemplating expanding the webscraping to ESPN since their API documentation is closed source :(.

Where is it?

It's not here... yet. Integrating Consolidate into this website gives me the perfect opportunity to learn and play around with the REACT JavaScript framework. I don't know REACT currently. But I want to, because it's cool and awesome and maybe i'll be able to impress my friends or a passing recruiter.
